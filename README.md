# MyBalance Mobile

Aplicación móvil creada con Expo + TypeScript.

## Requisitos

- Node.js 18+
- npm
- Android Studio (para emulador Android)

## Instalación

```bash
npm install
```

## Ejecutar en Android Emulator (Android Studio)

1. Abre Android Studio y asegúrate de tener instalado **Android SDK** y **Android Emulator**.
2. Ve a **Tools > Device Manager** y crea un dispositivo virtual (AVD) si aún no existe.
3. Inicia el emulador desde el Device Manager.
4. En otra terminal, inicia Expo:

```bash
npm run start
```

5. Cuando aparezca el menú de Expo, presiona `a` para abrir la app en el emulador Android.
